Diatonic: Of or using only the seven tones of a standard scale without chromatic alterations.
Triad: A set of three notes that can be stacked in thirds
Diatonics for all scales*:
I, II, III, IV, V, VI, VII
i, ii, iii, iv, v, vi, vii
*Uppercase numerals indicate a major chord; lowercase is for minor chords. This program only focuses on the major scales.
Major Scales: I, ii, iii, IV, V, vi, vii°
Minor Scales: i, II, III, iv, v, VI, VII/vii°
In music, a chord is a set of two or more notes played at the same time that follow a set of intervals in a certain key.

The most basic chord is called a Triad, which consists of three notes.

To create a Chord, press the buttons below the staff. These buttons correspond to notes on the staff.

When three buttons are pressed, the program will tell you some things. It will tell you the name of the chord, whether it is major or minor,

and the scale degree of the root of the chord, including the diatonics.

If you want to create another chord, refresh the page. There are also buttons that will change the key.
