This program allows people to enter notes on a staff to identify a chord.

With the help of JavaScript, there will be an alert when a chord is entered on the staff.

Chord Creator is powered by VexFlow, which is a JavaScript library for putting notes on a staff.

Chord Creator is tailored in order for the user to make different chords in a few keys.

Click here for directions of how to use Chord Creator, and some related music theory.
